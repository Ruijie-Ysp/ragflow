## 🔍 故障排查

### 问题 1: 构建失败

**错误信息**：
```
ERROR: failed to solve: failed to compute cache key: failed to calculate checksum
```

**解决方法**：
```bash
# 清理 Docker 缓存
docker system prune -a

# 重新构建
cd docker
docker-compose build --no-cache ragflow-proxy
```

### 问题 2: 容器无法启动

**检查日志**：
```bash
cd docker
docker-compose logs ragflow-proxy
```

**常见原因**：
1. 端口 10101 被占用
   ```bash
   # 检查端口占用
   lsof -i :10101
   
   # 修改 .env 中的 PROXY_PORT
   ```

2. RAGFlow 服务未启动
   ```bash
   # 启动 RAGFlow
   docker-compose up -d ragflow
   ```

### 问题 3: 健康检查失败

**检查容器内部**：
```bash
docker exec -it ragflow-openai-proxy curl http://localhost:8000/health
```

**检查环境变量**：
```bash
docker exec -it ragflow-openai-proxy env | grep RAGFLOW
```

### Docker Compose 配置

```yaml
ragflow-proxy:
  build:
    context: ..
    dockerfile: docker/Dockerfile.proxy
  image: ragflow-openai-proxy:latest
  container_name: ragflow-openai-proxy
  ports:
    - ${PROXY_PORT}:8000
  environment:
    - RAGFLOW_BASE_URL=http://ragflow:80
    - RAGFLOW_API_KEY=${RAGFLOW_API_KEY}
```
